# 📄 電子公文歸檔自動化系統 (Official Document Auto-Archiver)

這是一個專為公務人員與教育機構行政同仁設計的「電子公文自動歸檔小幫手」。透過 Python 與 Selenium 技術，協助使用者從繁瑣的公文歸檔點擊流程中解放，實現「批次選取、自動歸檔」。

## ✨ 核心特色

- **一鍵啟動專屬環境**：自動開啟獨立的 Chrome 瀏覽器分身，不影響日常上網記錄。
- **直覺式 GUI 介面**：使用 Tkinter 打造簡單明瞭的操作視窗，輕鬆勾選公文分類。
- **智慧分類補全**：只需輸入分類代碼數字（如 `03750401`），程式會自動比對內建辭典並顯示完整名稱。
- **全自動流程操作**：自動勾選目標公文、點擊「存查」、選擇分類、填寫案號並送出。
- **相容硬體憑證**：內建長效等待機制，完美包容「自然人憑證」或「組織憑證」的驗證時間。
- **自定義分類檔**：支援 `categories.txt` 設定檔，讓您根據業務需求自訂常用的分類清單。

## 🛠️ 環境準備

1. **安裝 Chrome 瀏覽器**：請確保電腦已安裝 Google Chrome。
2. **安裝 Python 環境**（若執行原始碼）：
   - 建議版本：Python 3.8+
   - 必要套件：`pip install selenium`
3. **下載 ChromeDriver**：程式會嘗試透過 Selenium 自動管理，若失敗請確保有對應版本的 ChromeDriver。

## 🚀 如何使用

### 使用執行檔 (.exe)
1. 雙擊執行 `公文自動歸檔.exe`。
2. 程式會自動開啟一個新的 Chrome 視窗。
3. **在該視窗中**：登入您的電子公文系統。
4. **進入待辦清單**：請切換到「待結案」畫面。
5. **回到程式視窗**：點擊「我已準備好，開始抓取清單！」。
6. **選擇分類**：針對抓取到的公文，從下拉選單選擇對應的歸檔分類。
7. **開始歸檔**：點擊「確認分類，開始批次自動歸檔！」，接著就可以放開雙手，讓程式自動運作。

### 使用原始碼 (.py)
```bash
python auto_archive.py
```

## ⚙️ 進階設定：自訂分類檔

程式啟動時會自動在同目錄下產生 `categories.txt`。您可以使用記事本開啟它，輸入您常用的分類代碼：

```text
# 每行一個代碼即可
03750401
03750402
03010203
```
程式下次啟動時，下拉選單就會顯示這些代碼對應的完整名稱，方便快速選取。

## ⚠️ 注意事項

- **安全性**：本程式僅模擬人工點擊，不儲存任何帳號密碼或憑證資訊。
- **視窗置頂**：程式視窗預設置頂，方便您在瀏覽器與操作介面間切換。
- **錯誤處理**：若歸檔過程中發生意外，程式會嘗試跳過該筆並繼續處理下一筆，並在最後輸出結果。
